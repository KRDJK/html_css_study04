# class
## class는 여러 개를 사용해서 별명을 붙일 수 있다. (ex 너는 참새이면서, 짹짹이면서 새대가리야)
## class 이름을 같게 하면서 서로 같은 그룹임을 표시할 수 있음.
## class="참새 짹짹이 새대가리"
